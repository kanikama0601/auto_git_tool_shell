# 使い方
## service_maker.sh
### 0.sudoで起動
```
sudo bash service_maker.sh  
```
一部sudoを使うコマンドがある為、sudoで起動してください。  
### 1.レポジトリ名入力
### 2.ユーザー名入力
### 3.グループ名入力
### 4.ターゲットディレクトリの絶対パスを入力
### 5..serviceのファイルを確認
### 6..shのファイルを確認
### 7.システムが起動してるかを確認
